Redis三种特殊数据类型

Nov 25, 2020点击

三种特殊的数据类型

geospatial 地理位置

朋友的定位，附近的人，打车距离计算。

Redis的Geo在Redis3.2版本就推出了。可以推算地理位置信息，两地之间的距离，方圆几里的人。

http://www.jsons.cn/lngcode/(查询中国城市经纬度)

六个命令

  • GEOADD
  • GEODIST
  • GEOHASH
  • GEOPOS
  • GEORADIUS
  • GEORADIUSBYMEMBER

Hyperloglog

基数：不重复的元素，可以接受误差

简介

Redis 2.8.9 更新了 Hyperloglog数据结构

优点：占用的内存是固定的，2的64次方，只需要12kb内存

网页的uv（一个人访问网站多次，但是统计还是算作一个人）

传统的方式，set保存用户id ，然后就可以统计set中的元素数量作为标准。

但是Hyperloglog有0.81%误差率

如果允许容错可以用Hyperloglog

Bitmaps

位存储

如果是统计人数： 0 1 0 0 0 1 0 0 (如果活跃就用1 不活跃就用0代表)只有两个状态的用bitmaps

Bitmaps位图，数据结构 都是操作二级制来进行记录，就只有0和1两种状态

例子

记录一周的打卡记录，1打卡 0未打卡

查看某天有没有打卡
